# PYTHON DJANGO TODO APP WITH AUTHENTICATION
By using this project, the user can manage his time and distribute plans.

## Installation
```
pip install django
pip install psycopg2
```

## Usage
```
django-admin.py startproject theproject
python manage.py runserver
```
